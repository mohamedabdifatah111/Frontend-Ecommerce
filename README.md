# 🛒 E-Commerce Frontend (React)

A fully responsive e-commerce frontend application built with React.
This project simulates a basic online store experience with product browsing, cart functionality, and checkout UI.



## 🚀 Features

* Responsive mobile-first layout
* Product listing page
* Product details page
* Add to cart
* Update and remove cart items
* Cart total calculation
* Cart persistence using localStorage
* Basic checkout form (UI only)
* Client-side routing with React Router



## 🛠 Tech Stack

* React
* React Router
* Context API
* CSS (Flexbox + Grid)
* LocalStorage

## 📦 Installation

Clone the repository:


git clone <your-repo-link>
cd ecommerce-frontend
npm install
npm run dev
```
